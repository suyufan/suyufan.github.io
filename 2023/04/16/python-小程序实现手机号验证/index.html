6.完善验证码+手机号存储（使用Redis）

  • Redis安装，启动。见链接

    • cd redis
    • ifconfig，查看服务器地址
    • 运行，src/redis-server redis.conf
      • 如果出现端口占用的报错：Creating Server TCP listening socket *:6379: bind: Address already in use
      • 运行ps -ef | grep -i redis，查看任务进程
      • 执行kill -9 进程号，终止进程
      • 重新启动

  • 在setting中配置，加上以下代码

  • pip install django-redis
