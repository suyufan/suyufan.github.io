vscode编写vue3快捷键

发表于 2021-01-10 | 分类于 工具使用 |
字数统计: 86 | 阅读时长 ≈ 1

创建快捷键，在VS code输入vv3回车即可自动vue3文档模板
html文档是!回车
步骤是：文件 - - - 首选项 - - -用户片段 - - - 在输入框中输入vue - - - 进入vue.json文件

代码: 

"Print to console": {
            "prefix": "vv3",
            "body": [
                "<template>",
                "</template>",
                "<script lang=\"ts\">",
                "import { defineComponent } from 'vue';",
                "export default defineComponent({",
                "\tname:'App',", 
                "});",
                "</script>",
            ],
            "description": "Print to console"
        }
